MIPS Emulator

A JavaScript based MIPS emulator that I created in the fall of 2025 while taking my computer architecture class. This project is currently unfinished, but is able to parse MIPS instructions by type and perform all the functions of the ALU. My end goal for this project is for user to be able to paste in a set of instructions and the emulator will parse them, display the hex translation and breakdown, and then highlight the different paths the data flows through. link

Medtronic Internship

The goal of my summer internship was to experience the full software development life cycle through the development of multiple features in the complete rewrite of an internal debugging website. This included acting as a requirements engineer and conducting stakeholder interviews to get an understanding of the pain points with the product. Next, I made backend changes using Octopus, SQL, Swagger, Restful API, C#, Entity Framework, and .NET. Then I did the front-end design work and implemented it using Angular, C#, HTML, and SCSS. I also did all the unit and integration tests for my work achieving over 99% code coverage.

Internet of Things (IoT)

To complete the ENGR 432 course focused on real time systems, this lab required the culmination of all the skills and technologies covered throughout the course. This included creating an Internet of Things (IoT) wireless and wired system that utilized RTOS, CubeMX, Arduino, Linux Commands, Python, two STM Boards, a Raspberry Pi 5, two WiFi 8266 modules, a HC-05 Bluetooth module, a photoresistor, a MCP9700A temperature sensor, UART communication of ADC readings, and a locally hosted website to graph the readings in real time using HTML/CSS and JavaScript.
